Detailed Description

The Workspace web service contains methods for creating, managing, querying and deleting workspace items. A workspace item can be considered a node which has not yet been completed and/or assigned ot a Path. Workspace items can refer to any object identifiable by a URI and most commonly references records from the Items table.
